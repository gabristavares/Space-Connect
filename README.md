# Space-Connect
O Space Connect é um projeto desenvolvido para a Global Solution 2026/1 da FIAP, com o objetivo de apresentar uma solução digital voltada para novos negócios, tecnologia espacial, dados orbitais e inteligência artificial.

A plataforma busca simular como empresas podem utilizar informações de satélites e análise de dados para melhorar conectividade, operação, tomada de decisão e impacto em diferentes regiões.

## 📌 Objetivo do Projeto

O projeto foi criado para oferecer uma experiência front-end moderna, responsiva e intuitiva, apresentando uma proposta tecnológica capaz de conectar empresas a soluções baseadas em dados espaciais.

A ideia principal é demonstrar como uma plataforma pode analisar informações de uma empresa e gerar recomendações personalizadas por meio de um simulador interativo.

## 🖥️ Funcionalidades

- Página inicial com apresentação do projeto.
- Menu de navegação entre todas as páginas.
- Logo clicável para retornar ao início.
- Página sobre com explicação da proposta e relação com ODS.
- Página de solução com etapas de funcionamento.
- Simulador interativo com cálculo de pontuação.
- Página FAQ com perguntas expansíveis.
- Formulário de contato com validação em JavaScript.
- Mensagem visual de sucesso ou erro ao enviar contato.
- Layout responsivo para desktop, tablet e celular.
- Animações em cards, botões e elemento visual da home.

---

## 📱 Responsividade

O site foi desenvolvido para se adaptar a diferentes tamanhos de tela, oferecendo boa experiência em:

- Desktop
- Tablet
- Celular

Foram utilizadas media queries no CSS para reorganizar menu, cards, formulários e seções, evitando cortes de conteúdo e rolagem horizontal.

## 🧭 Páginas do Projeto

- **Início:** apresentação do Space Connect, chamada principal e cards de impacto.
- **Sobre:** explicação da abordagem, ODS, dados e referências.
- **Solução:** descrição da proposta e etapas de funcionamento.
- **Simulador:** formulário que gera uma recomendação personalizada.
- **FAQ:** principais dúvidas sobre o projeto.
- **Contato:** formulário funcional com validação.
- **Integrantes:** informações dos autores do projeto.
